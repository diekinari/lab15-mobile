# QR Code Scanner

Android-приложение на **Java**: сканирование QR-кодов с помощью камеры. Результат отображается в диалоговом окне с возможностью открыть ссылку в браузере.

**Практическое занятие №15**, Тема 5 — Мобильная разработка.

---

## Стек

| Компонент | Версия |
|-----------|--------|
| Язык | Java |
| UI | XML layouts (ConstraintLayout) |
| Сканер | ZXing (zxing-android-embedded 4.3.0) |
| AGP | 8.7.3 |
| compileSdk | 34 |
| minSdk | 24 |

---

## Как работает

1. На экране одна кнопка — "Сканировать QR-код"
2. При нажатии запускается камера-сканер (ZXing)
3. При распознавании кода — диалоговое окно с содержимым
4. Кнопка "Открыть сайт" — открывает ссылку в браузере
5. Кнопка "OK" — закрывает диалог
6. Если код не распознан — Toast "QR-код не распознан"

## Как собрать

1. Открой проект в Android Studio
2. **File — Sync Project with Gradle Files**
3. Запусти на реальном устройстве (камера нужна для сканирования)
4. Разреши доступ к камере при первом запуске
